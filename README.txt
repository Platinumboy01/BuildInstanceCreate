PONGZ BlueStacks Multi-Instance Test Kit

>>> Read  SETUP-GUIDE.txt  for full setup & usage instructions. <<<

Quick version:
  1. Create instances in BlueStacks Multi-Instance Manager.
  2. Run  1-Enable-ADB-FirstTime.bat   (once per PC).
  3. Put your .apk in the  Builds  folder.
  4. Launch-All-Instances.bat   to open everything.
  5. Install-Build-To-All.bat   to install the build on all instances.
